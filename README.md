## 📋 Descrição

Este é um projeto de portfólio coletivo desenvolvido pelo **Grupo A**, showcasing os talentos e projetos de cada membro da equipe. O site apresenta uma interface moderna e responsiva, com design clean e funcionalidades interativas.

## 🚀 Funcionalidades

- ✅ Design responsivo para todos os dispositivos
- ✅ Menu de navegação interativo
- ✅ Cards individuais para cada membro
- ✅ Animações suaves e gradientes modernos
- ✅ Layout flexível e adaptável
- ✅ Efeitos hover nos elementos

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização e animações
  - Flexbox
  - Grid Layout
  - Animações CSS
  - Media Queries
